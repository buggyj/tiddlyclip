Tiddlyclip Browser Addon

Tiddlyclip (TC) allows parts of webpages to be clipped into a tiddlywiki (TW), and consists of two parts, the browser Addon and the TW plugin. This is the project page for the Addon part. The Addon is completely memoryless, any configuration is determined by the current TW that the user has select to work with. TC is designed to work (in a basic mode) without configuration.
